# Bases-de-datos
Para crear la base de datos: <br>
CREATE TABLE `usuario` ( `id` INT(10) NOT NULL PRIMARY KEY, `nombre` VARCHAR(20), `apellido` VARCHAR(20), `email` VARCHAR(20));
